# PythonMusicPlayer
This Python script serves as a music playing app and includes a console GUI.

NOTE:
This was an assignment final for my collegiate python programming course.

# Overview
PythonMusicPLayer uses a json file to load details about music and PyGame to play the music.
It includes many controls such as adding or playing songs and albums. 

# Technology Used
- Python 3.11
- PyGame
- json
